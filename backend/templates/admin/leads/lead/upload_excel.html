Excel File Format Requirements:

Your Excel file must contain the following columns (in any order):

  • Agent - Name of the agent (will be matched to existing users)
  • Date & Time - When the lead was created (format: DD.MM.YYYY HH.MMpm)
  • Customer Name - Full name of the customer
  • Customer Number - Phone number
  • Customer address - Street address
  • Customer Postcode - Postal code
  • Contact Centre Notes - Notes from contact centre
  • Outcome - Lead outcome (On hold, Pass Back, Blow Out, etc.)
  • Kelly Notes - Additional notes from Kelly
  • Data Source - Source of the lead data

Important Notes:

  • Phone numbers must be unique - duplicates will be skipped
  • Agent names will be matched to existing users by username or first name
  • If an agent is not found, leads will be assigned to the admin user
  • Outcome will be mapped to lead status automatically
  • Maximum file size: 10MB
